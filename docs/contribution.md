# Contributing to ZenBudget

Thank you for wanting to contribute. ZenBudget is a privacy focused finance tracker and contributions are welcome.

## Steps
1. Fork the repo
2. Clone locally
3. Install Flutter
4. Run flutter pub get
5. Create a new branch
6. Make changes
7. Submit a pull request

## Coding Guidelines
- Use Riverpod or Bloc
- Follow folder structure strictly
- Keep UI separate from logic
- Write small focused widgets
- Maintain clean data models

## Pull Request Requirements
- Clear title
- Description of changes
- Screenshots if UI related
- Link to issues if relevant
