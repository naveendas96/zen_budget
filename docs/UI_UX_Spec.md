# ZenBudget — UI/UX Specification

## 1. Design Philosophy
- Calm and Zen-like  
- Minimal, thoughtful spacing  
- Soft shadows and gradients  
- Material You components  
- Accessible & intuitive  

---

## 2. Color System
Primary: Zen Green (#3BAF79)  
Secondary: Sage (#A7D7C5)  
Light Background: #F9FAF7  
Dark Background: #0D0D0D  
Surfaces: #F2F2F2 / #1C1C1C  

Accent Colors:
- Lavender  
- Aqua  
- Teal  
- Gold  
- Sunset Orange  

---

## 3. Typography
Font: Inter / Google Sans  
- H1 → Balance numbers  
- H2 → Section titles  
- Body → Transactions  
- Caption → Dates  

---

## 4. Navigation

### Bottom Navigation:
- Home  
- Transactions  
- Budgets  
- Settings  

### FAB:
- Add Expense / Income / Transfer  

---

## 5. Screens

### Home
- Total balance  
- Budget summary  
- Insights carousel  
- Recent transactions  
- Upcoming subscriptions  

### Transactions
- Grouped list  
- Filters / search  
- Edit/delete  

### Add Transaction
- Amount field  
- Type selector  
- Category grid  
- Account / Payee  
- Notes / tags  

### SMS Pending Queue
- Inbox-style cards  
- Confirm/edit  
- Rule preview  
- Bulk actions  

### Budgets
- Monthly overview  
- Per-category details  
- Learning mode  
- What-if sliders  

### Subscriptions
- Auto-detected list  
- Due reminders  
- Payment history  

### Sync
- Manual/Auto backup  
- Restore list  
- Device linking  

### Settings
- Accounts  
- Categories  
- Rules  
- Themes  
- Privacy  
- AI preferences

## 6. Interactions
- Spring FAB  
- Smooth scroll  
- Card fade-in  
- Budget ring animation  
- Swipe to refresh

## 7. Accessibility
- Large text mode  
- High contrast themes

- Keyboard navigation where possible
