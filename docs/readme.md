# ZenBudget

ZenBudget is a privacy focused, offline first, AI assisted personal finance tracker built for India. It supports SMS parsing, advanced budgets, subscriptions, encrypted Google Drive sync, and a Zen inspired Material You UI.

## Features
- SMS based auto detection
- Pending Review Queue
- Budgets with learning mode
- What-if sliders
- Subscriptions detection
- AI assistant for insights
- Local encrypted storage
- Google Drive encrypted backup
- Full customization of categories and accounts

## Tech Stack
Flutter  
Riverpod  
Material You  
Drift or Isar  
Google Drive API  
AES encryption
